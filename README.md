# 🔍 Customer Churn Analysis — SaaS Product

## 📋 Project Overview
Analyzed customer churn patterns for a SaaS telecom product 
using real-world data of 7,043 customers to identify key churn 
drivers and provide data-backed business recommendations.

## 🛠️ Tools Used
- PySpark — data processing and SQL analysis
- Python (Pandas, Matplotlib) — EDA and visualization
- Databricks — cloud notebook environment

## 📊 Key Findings
- Overall churn rate: 26.5% (1,869 out of 7,043 customers)
- Month-to-month customers churn at 43% vs 3% for 2-year contracts
- New customers (0-1 year) churn at 47.4% vs 9.5% for 4+ year customers
- Customers without Online Security churn at 41.8% vs 14.6% with security
- Churned customers pay avg $74/month vs $61 for retained customers

## 💡 Business Recommendations
1. Convert month-to-month customers to annual contracts with 20% discount
2. Build first-year retention program — reduce new customer churn from 47% to 30%
3. Bundle Online Security and TechSupport as free add-on for first 3 months
4. Introduce mid-tier pricing plan at $55-65/month for price-sensitive customers

## 📈 Expected Business Impact
Full implementation of recommendations projected to reduce churn 
by 40-50%, saving $55,000-$70,000 in monthly recurring revenue.
